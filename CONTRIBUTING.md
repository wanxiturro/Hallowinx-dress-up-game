## 🚀 Cómo contribuir

1. **Fork** el repositorio
2. Crea una rama (`git checkout -b feature/nueva-funcionalidad`)
3. Haz tus cambios
4. Commit (`git commit -m 'Añade nueva funcionalidad'`)
5. Push (`git push origin feature/nueva-funcionalidad`)
6. Abre un **Pull Request**
